# Инструкция по загрузке фотографий для услуг

- ✅ `trade-in.jpg` - Trade-in (уже загружен)

---

## Как загрузить фотографию для Детейлинга

### Вариант 1: Через терминал (если у вас есть доступ к файлам локально)

1. Подготовьте фотографию детейлинга (рекомендуемый размер: **800×500px** или соотношение 16:10)
2. Переименуйте файл в `detailing.jpg`
3. Скопируйте файл в папку проекта:
   ```bash
   cp /путь/к/вашей/фотографии.jpg /Users/nasty/Projects/amgautotrade/images/services/detailing.jpg
   ```

4. Добавьте файл в git и закоммитьте:
   ```bash
   cd /Users/nasty/Projects/amgautotrade
   git add images/services/detailing.jpg
   git commit -m "Add detailing service photo"
   git push
   ```

### Вариант 2: Через GitHub (если у вас нет доступа к терминалу)

1. Откройте репозиторий на GitHub: https://github.com/n4styaparfenova/amgautotrade
2. Перейдите в папку: `images` → `services`
3. Нажмите кнопку **"Add file"** → **"Upload files"**
4. Загрузите файл с именем **`detailing.jpg`**
5. Добавьте описание коммита: "Add detailing service photo"
6. Нажмите **"Commit changes"**

### Вариант 3: Временная заглушка (если фото пока нет)

Если фотографии для детейлинга пока нет, можно:
1. Использовать временную заглушку (placeholder)
2. Или использовать одну из существующих фотографий

---

## Требования к фотографиям

- **Формат**: JPG (рекомендуется)
- **Размер**: оптимально 800×500px (соотношение 16:10)
- **Вес файла**: до 500 КБ (для быстрой загрузки сайта)
- **Содержание**:
  - **Детейлинг** - чистый салон автомобиля, процесс полировки, или результат детейлинга
  - **Trade-in** - уже загружен

---

## Проверка после загрузки

После загрузки фотографии проверьте:
1. Откройте сайт: https://n4styaparfenova.github.io/amgautotrade/
2. Найдите блок "Наши услуги"
3. Убедитесь, что фотография отображается в карточке "Детейлинг"
